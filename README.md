# Homelab Infrastructure (Proxmox)

## Overview

This project documents a self-hosted homelab environment running on Proxmox.

The setup includes network services, remote access, and home automation.

---

## Technologies Used

- Proxmox (virtualization platform)
- Pi-hole (DNS filtering)
- Tailscale (secure remote access VPN)
- Home Assistant (home automation)

---

## Architecture

- Proxmox host running multiple virtual machines
- Pi-hole used as DNS server for network-wide ad blocking
- Tailscale used for secure remote access
- Home Assistant for automation and device control

---

## Features

- Network-wide ad blocking
- Secure remote access without port forwarding
- Virtualized services on a single host
- Centralized home automation system

---

## Use Cases

- Secure remote access to home network
- Blocking ads and tracking domains
- Managing smart home devices
- Running multiple services in isolated environments
